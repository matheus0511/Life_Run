# Guia Visual Inicial - LifeRun

## Direcao visual

LifeRun deve comunicar energia, confianca e clareza operacional. A interface deve ser direta para atletas e eficiente para organizadores.

## Paleta

- Verde principal: `#1F8A4C`
- Verde escuro: `#126337`
- Amarelo de acento: `#F0B429`
- Azul informativo: `#2364AA`
- Texto principal: `#17211B`
- Texto secundario: `#607067`
- Fundo claro: `#F3F6F1`
- Superficie: `#FFFFFF`
- Borda: `#DBE4DC`

## Tipografia

- Fonte preferencial: Inter.
- Fallback: Segoe UI, Arial, sans-serif.
- Titulos devem ser fortes e objetivos.
- Textos de apoio devem ser curtos, com foco na acao do usuario.

## Componentes base

- Topbar com marca e navegacao principal.
- Botoes primarios para acoes de continuidade.
- Botoes secundarios para navegacao auxiliar.
- Cards apenas para itens individuais, resumos e conteudos repetidos.
- Secoes de pagina sem cards aninhados.

## Responsividade

- Layout maximo centralizado em `1120px`.
- Quebra para duas colunas abaixo de `900px`.
- Uma coluna abaixo de `620px`.
- Botoes ocupam largura total no mobile quando necessario.

## Acessibilidade

- Usar elementos semanticos.
- Manter contraste adequado.
- Definir textos de `aria-label` quando a funcao nao for obvia.
- Preservar foco visivel em links e controles.
